# Beauty Pro Inventory System

A professional, interactive inventory management system specifically designed for beauty entrepreneurs, makeup artists, and cosmetics retailers.

## Project Structure

```
inventory_template/
├── README.md                    # Project documentation
├── scripts/                     # Google Apps Script files
│   ├── Code.gs                 # Main automation functions
│   ├── Dashboard.gs            # Dashboard and analytics
│   ├── Setup.gs                # Guided setup functionality
│   └── Validation.gs           # Data validation functions
├── sheets/                     # Individual worksheet templates
│   ├── Dashboard.csv           # Main dashboard template
│   ├── Categories.csv          # Categories setup
│   ├── Suppliers.csv           # Suppliers management
│   ├── Products.csv            # Product catalog
│   ├── Inventory.csv           # Live inventory tracking
│   ├── QuickAdd.csv            # Quick entry interface
│   ├── Reorder.csv             # Reorder dashboard
│   ├── Analytics.csv           # Business analytics
│   └── Instructions.csv        # User guidance
├── templates/                  # Complete system templates
│   ├── BeautyPro_Starter.xlsx # Starter edition template

    └── sample_data.json        # Demo business data
